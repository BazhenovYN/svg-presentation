How to Embed SVG into HTML?

1. Embedding SVG as images

1.1 Using an <img> tag

						
<img src="path/to/svg/logo.svg" alt="logo" style="fill: lime;">

1.2 Using a CSS property background-image

						
.icon {
	background-image: url(logo.svg);
}

.icon:hover {
	background-color: lime; /* it does't work */
}

1.3 Using a CSS property mask-image

						
.icon {
	background-color: grey;
	-webkit-mask-image: url(logo.svg);
	mask-image: url(logo.svg);
}

.icon:hover {
	background-color: lime;
}

2. Embedding SVG as inline element

2.1 Using inline SVG in HTML5

						
<body>
	<svg viewBox="0 0 24 24" width="24" height="24">
		<!-- paths, shapes, etc. -->
	</svg> 
</body>

2.2 Using an SVG sprites

						
<svg style="display:none">
	<symbol id="example" viewBox="0 0 24 24" width="24" height="24">
		<!-- paths, shapes, etc. -->
	</symbol>
</svg>

<svg><use href="#example"/></svg>
